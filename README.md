## Description
### Weather Board is a search application that allows users to search for the current weather as well as a five day forecast in any city they like.
![Enhibition](https://media.giphy.com/media/yD0KZFWntDAEy5kgDJ/giphy.gif)
## Installation
### Simply navigate to the url and enter the city you would like to get information for into the search bar.
## Technologies Used
### HTML5, Bootstrap CSS Framework, Javascript, Jquery, Open Weather Map API
